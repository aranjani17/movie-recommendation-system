# 🎬 Movie Recommendation System

A simple and interactive Movie Recommendation System built using Python and Streamlit. This application suggests movies to users based on their selection from a list of available movies.

---

## 🚀 Features

- 🎯 User-friendly interface using Streamlit  
- 📋 Dropdown to select movies  
- ⚡ Instant movie recommendations  
- 📊 Uses real-world MovieLens dataset  
- 🚀 Fast and lightweight performance  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- Streamlit
